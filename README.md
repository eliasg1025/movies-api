# Movies
Base URI: `/api/movies`

End points: 

Name | Method | URI | Protected
-----|--------|-----|----------
Get movies | GET | /?tags | No
Get movie | GET | /:movieId | No
Create movie | POST | / | No
Update movie | PUT | /:movieId | No
Delete movie | DELETE | /:movieId | No
